# easing-functions

30+ funções de easing (ease-in, bounce, elastic)

## Instalação

```bash
npm install easing-functions
```

## Uso

```ts
import * as lib from 'easing-functions';
```

Veja `src/index.ts` para exemplos reais de cada função exportada.

## API

Todas as funções e tipos são exportados a partir do entrypoint principal. Consulte os tipos TypeScript gerados em `dist/index.d.ts` após o build para a referência completa.

Para rodar o build localmente:

```bash
npm install
npm run build
```
